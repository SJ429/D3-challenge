Discovered Between Health Risks and Age, Income

The purpose of this project is to use the D3 library of JavaScript to create scatter plots with data variables. The data include poverty, income, age and healthcare of all fifty states.

Therefore, a scatter plot is created to examine the relationship between income and poverty. The scatter plot shows the income level and poverty as a percent in each state.

Another scatter plot is created to examine the relationship age and healthcare. Does age determine heatlh hence how healthy a state?
